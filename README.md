# Vendor Performance Analysis

This project analyzes vendor and brand performance using Python and data analysis techniques.

## Objective
The goal of this project is to identify:

- Top performing vendors
- High profit margin brands with low sales
- Vendor contribution to total revenue
- Correlation between sales and profit

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statistical Analysis

## Key Insights
- A small number of vendors generate the majority of total sales.
- Some brands show **high profit margins but low sales**, indicating opportunities for promotions or pricing adjustments.
- Strong correlation exists between total sales and gross profit.

## Project Structure
Vendor_Performance_Analysis.ipynb → Main analysis notebook

## How to Run
1. Clone the repository
2. Install required Python libraries
3. Run the Jupyter Notebook

```bash
pip install pandas numpy matplotlib seaborn scipy
```
